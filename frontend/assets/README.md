# Assets Directory

This directory should contain the following image assets for the Expo app:

## Required Assets

1. **icon.png** (1024x1024)
   - App icon for iOS and Android
   - Should represent HeritageAI/cultural preservation theme

2. **splash.png** (1284x2778 for iPhone, or 1242x2436)
   - Splash screen image
   - Displayed when app is launching

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon foreground
   - Should work on various Android icon shapes

4. **favicon.png** (48x48 or larger)
   - Web favicon
   - Optional for web version

## Creating Assets

You can use tools like:
- Figma
- Canva
- Adobe Illustrator
- Online icon generators

## Placeholder

For now, you can use placeholder images or generate simple colored squares to get started. The app will work without these, but they're required for production builds.

## Asset Guidelines

- Use cultural/heritage-themed imagery
- Keep colors consistent with app theme (browns, beiges, warm tones)
- Ensure icons are clear and recognizable at small sizes
- Follow platform-specific guidelines (iOS Human Interface Guidelines, Material Design)
